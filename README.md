# SFE-Game-The-Great-Garbage-Avalanche-of-2505
(slop factory experiment - javascript game) Free our Heroes From their Hibernation Chambers

-----------------------------------------------
The-Great-Garbage-Avalanche-of-2505 is a 3D Javascript Game to Play in the Browser:

start of the game : the two hibernation chambers containing Rita & Joe are randomly placed inside an initial stack of garbage.

game play : one garbage is randomly generated at a time, user must put it somewhere on the garbage mountain.

goal : trigger a big enough avalanche so that the two hibernation chambers slide out of the garbage.
